Advanced open source package malware & vulnerability checker

About SafePack

SafePack checks packages against OSV.dev to identify malicious and vulnerable packages.

  • OSV.dev - Google's Open Source Vulnerability database, aggregating malware data from OpenSSF, GitHub Security Advisories, and more.

Note: A clean result doesn't guarantee safety. Always review package code and use packages from trusted maintainers.

Data sourced from osv.dev
